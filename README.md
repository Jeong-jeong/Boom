## 👀 프로젝트 빌드 및 실행 방법

1. 상단 `Code` 버튼을 눌러 레포지토리를 클론 받습니다.

```
$ git clone
```

2. 패키지를 설치합니다.

```
$ yarn install
```

3. Live Server 등의 플러그인을 통해 로컬에서 실행할 수 있습니다.
